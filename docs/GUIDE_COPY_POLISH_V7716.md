# v7.71.6 使い方ガイド文脈整理

## 目的

ガイド内で役割が重なっていた「対策ケア」と「記録・分析」を整理し、ユーザーが機能の違いを一目で理解できるようにする。

```text
トリセツ
  自分の体質を知る

体調予報
  今日・明日のゆらぎを見る

対策ケア
  先回りして整える

記録・分析
  ケアと実感から傾向を見る
```

## 主な変更

- `Daily Care` の表示を「対策ケア」へ統一
- ヒーローの4項目を、機能名ではなく利用目的で役割分担
- 記録・分析のアイコンを専用の鉛筆アイコンへ変更
- 「前日の明日カード」「昨晩の明日ケア」など不自然な表現を修正
- 対策ケアの実行記録と、夜の実感記録の関係を明確化
- 提案以外のケアは、従来のまとめ入力として記録することを明記
- AI・グラフのカードには分析用アイコンを使用
- metadataとHowTo構造化データも「対策ケア」へ統一

## DB・APIへの影響

なし。ガイド表示、アイコン、metadata、テストのみの変更。
