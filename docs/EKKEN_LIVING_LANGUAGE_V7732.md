# v7.73.2 Ekken 生活者向け翻訳と安全案内の反復抑制

## 背景

v7.73.1でライブ相談プロンプトのマイクロ管理を削り、Ekkenが体質・予報・現在の体感を自由に統合できるようにした。実機回答では内容の妥当性が大きく改善し、特に経絡プロフィールと現在症状を分けて重ねる回答ができるようになった。

一方、食養生・暮らし・漢方では「気虚」「気滞」「湿をさばく」「余力を守る」などの概念が専門語のまま出やすく、説明書のように見える場面が残った。また、同じ会話の中で漢方、暮らし、ツボへ話題を移すたびに、同じ専門家確認や受診目安が再掲されることがあった。

## 方針

### 東洋医学を生活者の言葉へ翻訳する

Ekkenは東洋医学の体系を内部の思考道具として正確に使う。そのうえで会話では、専門語を知らない人でも身体の中を想像できる現代語を優先する。

比喩は必須ではない。理解が深まる時だけ自然に使い、毎回答へ入れず、同じ比喩へ固定しない。

`product_context.communication_translation.sensitivity_examples` は、次のような翻訳の感度を示す作例であり、定型句や一対一の置換辞書ではない。

- アクセル優位×余力小: アクセルは踏めるけれど、長距離を走る燃料は少なめ
- 気滞: 力がないというより、出口付近で流れが渋滞している
- 痰湿: 体の中に湿った重い荷物が残り、動き出しに抵抗がかかる
- 血虚: 身体を養う材料のストックが少なめ
- 津液不足: うるおいを保つ貯水量が少なめ
- 経絡: 臓器の病気そのものではなく、不調が表れやすい体の道筋

「便」のように体調文脈で別の意味へ読める表現は避ける。

### 安全案内を同じ会話で繰り返さない

同じ会話ですでに伝えた一般的な注意、専門家確認、受診目安は、次の場合だけ再掲する。

- 新しい危険情報が加わった
- 状態が悪化した
- 判断に必要な条件が変わった
- 新しい質問へ直接必要な個別注意がある

相談テーマが漢方から暮らし、ツボなどへ変わっただけでは再掲しない。

直近会話のassistant messageには `safety_level` を含め、モデルが既出案内を認識できるようにする。さらに、サーバーが付加する固定の一般的専門家確認文は、同一会話ですでにprofessional案内があれば再付加しない。新しく必要になった個別の安全案内は引き続き表示できる。
