# 気圧の物理方向と体質反応の統合 v7.78.17

## 修正目的

v7.78.16までの予報計算は、気圧変化の大きさと体質による反応方向を分けていた。一方、計算後の一部処理では`pressure_down / pressure_up`を、そのまま体の表れ方やケアのキーとして使う旧契約が残っていた。

そのため、たとえば「気圧低下×アクセル反応」で、計算は張り・力み側を示すのに、文章や食事では重だるさ側を選ぶ可能性があった。本修正では、予報スコアを変えず、計算結果から表示・ケア・AI相談までの接続を一本化する。

## 1. データ契約

| 情報 | 値 | 用途 |
|---|---|---|
| `pressure_direction` | `up / down / mixed` | 天気カード、気圧アイコン、物理変化の説明 |
| `pressure_response_direction` | `accel / brake / balanced` | 不調サイン、ケア、食事、ツボ、Ekken、商品選定 |
| `personal_main_event_key` | 原則`pressure_shift` | 気圧を一つの変化イベントとして扱う識別子 |
| `personal_main_trigger_exact` | `pressure_up / pressure_down` | 既存保存形式との互換用の物理イベント |
| `body_response_key` | `pressure_accel / pressure_brake / pressure_balanced` | 体の表れ方を直接示す内部キー |

`personal_main_trigger_exact`だけから体の反応やケアを決めてはいけない。旧辞書を使う箇所は、`getLegacyCareTriggerKey`で反応方向を互換キーへ投影する。

## 2. 必須の4組

| 物理変化 | 体質反応 | 体の表れ方 | 天気カード |
|---|---|---|---|
| 気圧低下 | アクセル | 張り・力み、切り替えにくさ | 気圧低下 |
| 気圧低下 | ブレーキ | 重だるさ、動き出しにくさ | 気圧低下 |
| 気圧上昇 | アクセル | 張り・力み、切り替えにくさ | 気圧上昇 |
| 気圧上昇 | ブレーキ | 重だるさ、動き出しにくさ | 気圧上昇 |

`balanced`は、上昇・低下へ固定せず、環境変化への適応疲れとして扱う。

## 3. 下流処理

反応方向を次まで引き継ぐ。

- 予報本文、明日出やすいサイン、注意時間前の行動
- 東洋医学的な整理、母子選択、経絡・ツボ候補と選定理由
- 暮らす・食べる・ほぐすのDaily Care
- 今日のケア用の簡易ツボ・食事・過ごし方
- Ekkenの予報理解と相談入力
- パーソナルケアショップと提携商品の天気適合

旧辞書の`pressure_down`は重さ側、`pressure_up`は張り側の互換辞書として残る。ただし選択は物理方向ではなく反応方向から行い、本文中の「低気圧だから重い」「気圧上昇だから張る」という断定は、体の説明・ケアでは「気圧変化」へ中立化する。天気カードの上昇・低下表示は維持する。

## 4. 気温表示の役割

- 変化方向: `気温上昇 / 気温低下`
- 絶対環境: `高温 / 低温`
- 極端域の安全注意: `猛暑 / 酷暑 / 厳しい寒さ`

体質別予報と別に、次の独自注意を`environmental_cautions`へ出す。

- 最高気温35℃以上: 猛暑への注意
- 最高気温40℃以上: 酷暑への注意
- 最高気温0℃未満、または最低気温-5℃以下: 厳しい寒さへの注意

これは最高・最低気温だけによる未病レーダー独自の補助表示であり、公的な警戒アラートやWBGTではない。UIにもその旨を明記する。体質別ゆらぎ度の点数はこの表示追加によって変更しない。

## 5. 維持するもの

- v7.78.16の体調ゆらぎ度の式、快適域、重み、余力補正
- 気温・湿度・気圧の3負荷と高・中・低表示
- 今日01:05固定、明日16:30再計算
- 未ログイン予報とログイン予報の共通V2エンジン
- `RADAR_FORECAST_MODEL_VERSION=v1`による旧モデルへのロールバック
- 東洋医学の状態を専門知識なしで想像しやすくする、意味の通る比喩表現

## 6. 回帰確認

- 気圧の上昇・低下とアクセル・ブレーキの4組をすべて検証する
- 物理方向が同じでも、反応方向によりサイン・ケア・ツボ・商品選定が変わる
- 反応方向が同じなら、上昇・低下の違いだけで体の説明を反転させない
- 独自気温注意の有無が体質別ゆらぎ度を変更しない
- 35℃未満では猛暑注意を出さず、35℃・40℃・厳寒の境界値を固定する
