# 未病レーダー プロダクト方針

この資料は、未病レーダーの変わりにくいプロダクト方針をまとめたものです。  
現在の進行状況、完了済み項目、次にやる作業はここには書きません。

---

## コアメッセージ

> 明日の崩れやすさを、今夜の整え方に変える。

未病レーダーは、単なる「今日の不調を当てるアプリ」ではなく、明日の崩れやすさを先回りし、今夜から整えるための未病予報アプリです。

---

## 役割分担

### ダッシュボード

アプリを開いた瞬間の「今日ここからの伴走」を担当します。

```text
今日ここからの軽いリアルタイム予報
キャラクターによる一言コメント
明日の未病予報への導線
パーソナル体質トリセツへの導線
```

今日予報は、医療的な対処ではなく、今日ここから少し楽に過ごすためのヒントとして扱います。

### 予報ページ

明日の未病予報をしっかり見る本番ページです。

```text
明日の未病予報
アニメーションスコアゲージ
響きやすい要素
明日の山場
AI生成の読み解き文
今夜の先回りケア
  ほぐす / 食べる / 暮らす
```

「明日の山場」は端的に伝え、詳細な注意点や読み解きはAI生成文へ寄せます。

---

## 予報ロジック方針

```text
今日予報:
  AI生成なし
  DB保存なし
  開くたびに今以降だけで再計算
  ルールベースの軽い一言

明日予報:
  本命
  ルールベースでスコアと構造を決める
  GPT生成は読み解き文・ケア文の自然化に寄せる
```

GPTにスコア、主因、医学的判断を自由に再推論させない方針です。

## ケア提案UI

ケア提案は以下の3タブを基本にします。

```text
ほぐす
食べる
暮らす
```

### ほぐす

ツボケアを担当します。

```text
選ばれたツボ
どんなとき向きか
押し方
選定理由
```

### 食べる

食養生を担当します。

目指す方向:

```text
身体の読み解き
やりがちな失敗
意外な置き換え
避けたい組み合わせ
具体的な食べ方の作戦
```

一般的な健康アドバイスではなく、「この日の身体はどうなりやすいか」から食べ方へ落とし込みます。

### 暮らす

身の回りでできる東洋医学的ライフハックを担当します。

例:

```text
湿気の日の寝室や洗濯物の置き方
気圧低下の日の耳まわり、首肩、照明、音の扱い
冷え込みの日の足元、腹腰、寝具
乾燥の日の寝室、喉、湯気、加湿
暑湿の日の風の通り道、寝苦しさ対策
```

「無料でできる」のような開発者メモ的文言は避けます。

## アフィリ・物販導線

アフィリや物販は、広告枠ではなくケア提案の延長として扱います。

採用方針:

```text
人間が商品カタログを作る
商品やカテゴリにタグを付ける
体質・天気・時間帯とスコアリングで候補を選ぶ
AIは候補内で見せ方や説明を整える
```

避ける方式:

```text
湿気×冷えならA
湿気×低気圧ならB
気虚×冷えならC
```

採用する方式:

```text
カテゴリ/商品のタグ重み
×
予報・体質・時間帯のスコアリング
```

## PR表示方針

PR表示は必要ですが、広告枠っぽく見せない方針です。

避ける表現:

```text
おすすめ商品
今すぐ買う
効く
改善する
治る
痛みを和らげる
```

使いやすい表現:

```text
この日の支度
今夜の備え
暮らし方に合わせたアイテム
PRを含みます
```

効能断定ではなく、身体の読み解き・比喩・作戦名で伝えます。
