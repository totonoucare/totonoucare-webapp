# 体質トリセツ「整え方」コピー整理 v7.78.15

## 目的

体質トリセツの「整え方」タブを、7つのケア方針の目的が専門知識なしでも分かり、日々の体調予報との役割差も伝わる文章へ整える。

## 役割分担

- 体質トリセツ：体質チェックの結果から見た、ふだん合いやすい整え方を上位3件表示する。
- 体調予報：その日の天気と今気になる不調を重ね、当日に優先する整え方を選ぶ。
- パーソナルケアショップとEkken：同じ7方針を、商品選びと相談の共通語彙として使う。

## 表示文の基準

- 方針名だけでなく、何をどう整えるかを一文で示す。
- `逃げ道`や`余白`など、対象や目的が曖昧な内部設計語は避ける。
- `重だるさをためない`、`巡りを保つ`、`疲れを増やさない`など、体調予報の方針文と同じ平易な語彙を使う。
- 上位3件は優劣の断定ではなく、`基本にしたい／次に意識したい／補助として取り入れたい`と役割を示す。

## 非変更範囲

- 7方針の定義キー
- 体質、上位サブラベル、不調、環境回答から作るスコア
- 上位3件の選択順
- 体調予報のV2計算とケア選定
