# v7.71.5 使い方ガイド再構成

## 目的

機能追加によって分散していた説明を、ユーザーが実際に使う順番へ並べ直す。

```text
体質トリセツを作る
  ↓
今日・明日の体調予報を見る
  ↓
Daily Careを試して「やってみた」を押す
  ↓
夜に実感を記録する
  ↓
グラフ・AI分析で自分の傾向を振り返る
```

## 主な変更

- ガイドの主役を「トリセツ・予報・MYケア」の機能紹介から、予報→ケア→記録→分析の利用体験へ変更
- タブを「基本の流れ」「予報・ケア」「記録・分析」「MYケア」に再編
- Daily Careの実行記録、前夜ケアと当日ケアの扱い、夜の記録方法を追加
- 体調ゆらぎ度0〜100と、安定・いたわり・守りの役割を説明
- 統合グラフとAI分析の比較方法を追加
- 記録によって予報値を書き換えず、固定された予報を比較の物差しにする方針を明記
- 専門家相談を、AI分析から自然につながる補助機能として説明
- MYケアセレクトを毎日の必須導線ではなく、必要な時に使う補助機能として整理
- SEO metadataとHowTo構造化データを現行フローへ更新

## DB・APIへの影響

なし。画面文言とガイド構成のみの変更。
