# 未病レーダー 開発計画 v0.2 要約

最終更新: 2026-05-08

## コアメッセージ

> 明日の崩れやすさを、今夜の整え方に変える。

未病レーダーは、単なる「今日の不調を当てるアプリ」ではなく、明日の崩れやすさを先回りし、今夜から整えるための未病予報アプリとして設計する。

---

## 画面ごとの役割

### ダッシュボード

役割:

- 今日ここからの軽いリアルタイム予報
- キャラクターによる一言コメント
- 明日の未病予報への導線
- パーソナル未病カルテへの導線

今日予報は:

- AI生成なし
- DB保存なし
- MET Norwayから取得できる「今以降〜今日終わりまで」を使ってルールベース表示
- 医療的対処ではなく「このあと重くなりやすい時間」「今できる軽い一手」を見せる

### 予報ページ

役割:

- 明日の未病予報をしっかり見る本番ページ

構成:

```text
明日の未病予報
↓
アニメーションスコアゲージ
↓
響きやすい要素 1〜2個
↓
明日の山場
↓
AI生成の読み解き文
↓
今夜の先回りケア
  ほぐす / 食べる / 暮らす
```

「明日の山場」は時間帯を端的に伝える。  
余計なルール作文は入れず、詳細な注意点はAI生成文に任せる。

## ケア提案UI

カードを縦に増やすのではなく、以下のタブに分ける。

```text
ほぐす / 食べる / 暮らす
```

### ほぐす

既存のツボケア。  
内容:

- 選ばれたツボ
- どんなとき向きか
- 押し方
- 選定理由

### 食べる

既存の食養生カード。  
今後は「当たり前の健康アドバイス」から「身体の読み解き」に進化させる。

弱い例:

> 湿気の日は脂っこいものや甘いものを控えましょう。

目指す例:

> この日は、食べたものが力になる前に「重さ」として残りやすい日。冷たい麺と甘いカフェラテを重ねるより、温かい汁物に香味野菜を少し足す方が、流れを作りやすい。

### 暮らす

新設タブ。  
身の回りでできる東洋医学的ライフハックを担当する。

例:

- 湿気の日の寝室や洗濯物の置き方
- 気圧低下の日の耳まわり、首肩、照明、音の扱い
- 冷え込みの日の足元、腹腰、寝具
- 乾燥の日の寝室、喉、湯気、加湿
- 暑湿の日の風の通り道、寝苦しさ対策

「無料でできる」のような開発者メモ的文言は避ける。  
使うなら:

```text
今夜の支度
暮らしの一手
身の回りケア
```

## アフィリエイト・物販導線

採用方針:

- 「バーチャル養生キットカードを1枚追加」は採用しない。
- モバイルで縦長になりすぎるため。
- 広告カード感が出やすいため。
- 代わりに `ほぐす / 食べる / 暮らす` 各タブ内に自然にPR導線を差し込む。

最終形:

```text
人間が商品カタログを作る
↓
商品やカテゴリにタグを付ける
↓
体質・天気・時間帯とスコアリングで候補を選ぶ
↓
AIは候補内で見せ方や説明を整える
```

商品選定は if文分岐地獄にしない。  
タグ重み × 予報・体質・時間帯のスコアリングで、小さな推薦エンジンとして設計する。

PR表示方針:

- PR表示は必須。
- 広告枠っぽく見せない。
- 避ける表現: おすすめ商品 / 今すぐ買う / 効く / 改善する / 治る / 痛みを和らげる
- 使いたい表現: この日の支度 / 今夜の備え / 暮らし方に合わせたアイテム / PRを含みます

## 実装ステップ

### Phase 1: 明日予報UXの仕上げ

- ダッシュボード/予報ページの文言調整
- 響きやすい要素2チップ表示確認
- 暮らすタブのラベル調整
- 今日ここからカードの見た目調整

### Phase 2: ケア提案の質向上

- 食べるタブAIプロンプト強化
- 暮らすタブのライフハック拡充
- ほぐすタブの見せ方微調整

### Phase 3: アフィリ導線の器作り

- 各タブ下部にPR枠を差し込める構造を作る
- feature flagで制御
- 最初は商品リンクなし/仮リンクでも可

### Phase 4: 商品カタログと推薦エンジン

- 商品カテゴリ定義
- 商品カタログ定義
- タグ重みスコアリング
- 体質×天気×時間帯で候補選定

### Phase 5: クリック計測とD2C検証

- クリックイベント保存
- 体質/天気/カテゴリ別CTR分析
- 反応の良いカテゴリを養生キット候補化
- 将来の自社D2C/OEMへ接続
